About PramukhIME CKEditor Plugin

What is PramukhIME CKEditor Plugin?

PramukhIME CKEditor Plugin is a plugin for CKEditor HTML editor using which any Web developer can enable the users to type in 23 Indian languages without downloading/installing any special software. It supports Assamese, Bengali, Bodo, Dogri, Gujarati, Hindi, Kannada, Kashmiri, Konkani, Maithili, Malayalam, Meitei, Manipuri, Marathi (Devnagari and Modi), Nepali, Oriya, Punjabi, Sanskrit, Santali (Ol Chiki), Sindhi, Sora, Tamil and Telugu

Plugin Installation

  • Unzip the file on your local hard disk.
  • Upload/copy "pramukhime" folder to CKEditor_ROOT/plugins/ so the folder becomes CKEditor_ROOT/plugins/pramukhime/ (PRAMUKHIME_ROOT)
  • In the CKEditor configuration, add "pramukhime" in a plugin list and add "pramukhime" toolbar to show language drop down and other buttons

Plugin Customization

If you want to customize a list of languages to be displayed in dropdown list, change toggle key and for advanced options, add "pramukhime_options" parameter in CKEditor configuration. Follow the steps given in "example/example.html file
