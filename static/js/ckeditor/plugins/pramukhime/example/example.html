PramukhIME CKEditor Plugin Example



How to use?

  1. Download CKEditor 4.x (NOT 3.x) and unzip on hard drive (CKEditor_ROOT)
  2. Unzip PramukhIME CKEditor Plugin into "CKEditor_ROOT\plugins\" folder.
  3. Place this file example.html into "CKEditor_ROOT\samples" folder or change the path of "ckeditor.js" in this file.
  4. Open this file using real webserver like IIS/Apache and you should be able to view PramukhIME drop down list and buttons.
  5. Open this file in text editor and change the options given as per comments within html code to customize PramukhIME plugin.
